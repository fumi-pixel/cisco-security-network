# Tri-Zone Defence Network — Cisco ASA, DMZ & VLAN Segmentation

A security-focused network built in Cisco Packet Tracer featuring:
- Cisco ASA 5506-X firewall with three security zones
- DMZ with Web, DNS and FTP servers
- VLAN segmentation (Management, HR, IT)
- ACLs, NAT/PAT, and port-security

# Tools Used
Cisco Packet Tracer 8
